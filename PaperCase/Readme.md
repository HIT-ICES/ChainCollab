# PaperCase Directory

This directory contains example case studies, algorithms, and templates used in the paper.

---

## 📂 Example Case Studies
- `example/CaseStudy_ETDM.json` – Element Topology Description Model (ETDM) example  
- `example/CaseStudy_EDDM.json` – Element Data Description Model (EDDM) example  
- `example/CaseStudy_SDM_p1.json` and `example/CaseStudy_SDM_p2.json` – Statecharts Description Model (SDM) examples  
- `example/CaseStudy_SC.go` – Hyperledger Fabric Smart Contract example  

---

## ⚙️ Algorithm Parsers
- `Algorithm/parser_p1.py` – BPMN → ETDM parser  
- `Algorithm/parser_p2.py` – BPMN → EDDM parser  

---

## 📝 Templates
- `Template/BSAET.py` – BPMN-Statecharts Atomic Element Template (BSAET)  
- `Template/BST.json` – BPMN-SC Template (BST)  

---
